2019春节随感

发表于 2019-02-05 | 分类于 思考

忽忽然间，已经旧历春节了。

28岁的这一年，应当比较艰难。

回望来路，在D司的这两年，增了些气质，涨了些自信，绕了些弯路，蹉跎了岁月，徒增了年纪。

到头来，遇到一道坎：经历和实力不相符。须鲤鱼般跃过去！

17考研中山，虽此道遥通罗马，然终究犯了“销售需要揣摩他人心思去读心理学硕士” – 铺大贪多的毛病。

18做Android，第一季度摸索完成项目，第二季度开始规范流程、学习Java，第三季度切入P项目，难度和速度陡然增加，四季度方能按需完成。

从内往外看：1年iOS，1年考研夯实基础，1年Android。貌似没有浪费时间，只因站在此山中。

从外向内看：毕业五年，水准没有5年的精进。亟需快速追上。

向后规划5年，我需要有自己的事业。操盘事业需要哪些能力呢？

  1. 技术 为立身之。

    • 事业从工作周边出发，不能跳跃性太大。否则，之前的经历全部作废。

      类似阿里：淘宝/天猫 --> 支付宝 --> 蚂蚁金服
                    --> 阿里云
                            --> 菜鸟
                          --> 闲鱼
                       --> 大数据 --> AI

    • 技术小牛、业界10%水准。

      一般人职业路径是一定的级别转做管理，所以：1%需要天赋，但10%努力可得。

  2. 总结：框架流程图

    工作、技术、生活中遇到的问题，解决方案+思路+主要框架点+画流程图。

  3. 表达

    • 抓主干，理清流程图，放弃细枝末节
    • 找中心，用自己语言复述
    • 讲故事，锻炼语气
    • 思路+精准表达

  4. 降维打击

    • 使用管理思维思考团队遇到的问题，思考如何解决。
    • 此时，与leader站在同一高度。
    • 熟悉此高度需要做的事情，并熟记会遇到哪些方面的问题。
    • 遇到事情，检查各个方面是否有问题。方能快速抛出有用的问题。
    • 多看管理、架构、宏观方面知识，多总结关键点，并熟记。

  5. 气场+气质

    • 气场 –> 自信（技术、实力、财力、搞定经历）
    • 气质 –> 穿搭、锻炼
    • 锻炼 –> 跑步机+电视剧+5km+走路5分钟 –> 脑子清爽思路开阔

  6. 简单 极致

    • 不贪多，近几年的弯路：过于长远的考虑到10年后的生活，打乱了当下的生活。
    • 不能轻视手头的东西，做到极致，必然有路可走。

  7. 坚定乐观者，自我修复能力强

    遇到不顺心的事，尽快恢复理智，理性做任何决策。

  8. 自我控制

    • 自律即自由
    • 身体控制思想
    • 抛弃手机，信息爆炸时代，需要留白思考。
    • 合理规划时间
    • 合理规划效率
    • 知行合一，执行力

  9. 不同的场合，展现不同的个性。

    • 个性无对错，适应场合最重要。
    • 内向 –> 演讲能力
    • 不会说话 –> 针对性的搜集材料、练习、背诵、定向突破

  10. 情商，学做事。

    • 观察生活中的人如何处理问题，优缺点如何
    • 换位思考，自己如何做的更好。
    • 反思当下，自我环境，别人需求。

以上10条，可以作为当下的军规，无条件执行。

若每个阶段都能看到一定的成果，待大成之日，即为甘来之时。
