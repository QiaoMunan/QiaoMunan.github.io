周记001

口水能定汇率吗？

关于汇率问题，有人总是吵吵闹闹，说该升值的该贬值的都有。
但是市场有什么好吵的？直接出价就行了。
看多，你就用美元买人民币呀；看跌，你就卖人民币还美元。

97年，亚洲金融危机，国际：“人民币不要贬”
02年，国际：“人民币要升值”
蒙代尔说人民币不可升，但是一直待北京。
有朋友说人民币不应该升值的理由头头是道，但是又全部卖掉了美元。

形成人民币汇率的市场特征

中国外汇交易中心

  1. 中国外汇交易中心是一个建设在中国的市场，人民币的交易仅限于中国。之所以现今要求人民币升值的呼声嘹亮，是因为他们那里无从用美元买人民币。
  2. 中国外汇交易中心，实行会员制，会员指的是：在中国具有合法经营外汇业务的商业银行和其他金融组织。
  3. 所有进入中国市场的“热钱”，都是先与这些会员结汇，然后到中国外汇交易中心进行买卖、交易。所以，汇率其实就是交易中心的市场上进行竞争买卖的结果。
  4. 央行也可以进行购汇。虽然既当裁判，又当运动员，但是规则透明。

央行购买外汇的来源是：基础货币。

央行购汇的支付手段

  1. 增发人民币
  2. 商业银行的存款准备金
  3. 基准利率，加息减息。
  4. 发行央票回收流动性

央行负债的经济性质

央行的全部外汇储备，都对应着央行的人民币人民币负债。而且 央行欠债的时候，不需要得到债权人的同意，也不需要让债权人知道

央行总资产中，每1000元，其中999元来自于负债。一般人肯定不能借到这么多，为什么央行可以？因为央行是政府开的，只要大家相信政府财政能力，央行的资产负债就无需操心。

为什么央行的负债不需要归还？

  1. 发行的货币，全部是借条，亦就是央行的负债。
  2. 存款准备金 可以用增发货币来还。
  3. 债务到期以后，可以再发一部分，把上次的全部还掉。（类似于，地方债务、银行贷款）

因为负债不需要还，所以央行的负债资产可以接近无限，持有的 外汇资产也可以接近无限，这就是形成人民币汇率的基础。
所以，人民币汇率不是买卖行为的结果，而是各国政要专家舆论大打口水战的结果。

如此辛苦为哪般？

近十年来，央行不断忙一件事儿，就是以更高的资产负债率，大手笔买下十倍于前的外汇资产。

为什么这么做？央行到底图什么呢？ 肯定不图利。

  1. 因为不扩大举债，央行就没有能力在外汇交易中心，大手收购，越来越多的进入中国的外汇。
  2. 央行负债的一个大科目，就是储备货币。
  3. 维持人民币币值的稳定
  4. 央行要忙着回收流动性，发央票，增加法定储备金，提升利息，有时候还需要使用中国特色的其他政策工具。

总体上，来来回回就两件事：

  1. 大手举债购买外汇；
  2. 痛下决心，收回放出去的流动性
