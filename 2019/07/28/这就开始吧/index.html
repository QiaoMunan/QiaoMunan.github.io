就这样开始吧

发表于 2019-07-28 | 分类于 记一忘三二

打小记事起，我就知道自己不是一个聪明的人。
长相一般、不喜娱乐、言语无趣、说话易得罪人，活脱脱一个木讷呆板淌着鼻涕的小男孩。
好在性格乖巧，听大人的话认真学习；加上脑袋空灵，在西北黄土高原上一个三面环沟的小村子里成绩尚可。
成绩好理算当然的带给了我一系列的特权：奖状贴了一墙；村里的长辈夸“村里的独苗”；赢得同学尊重；在物质匮乏的年代搜集到金庸全集…如此种种不一而足。
成绩好带给我最大的优点就是–去探索来满足自己的好奇心：我深知自己不聪明，周围也大多蠢笨之人，只能靠书本来获取知识，完成自己的原始知识架构。

到了一个可以输出的年纪，回望自己走的踉踉跄跄的来路，复盘一下自己的历程，愿以后走的稳健。

1. 知识框架搭建成功

我的第一次升华在大学毕业后一两年。
某天突然发现，在生活中、社会上发生的事物，我都能用自己的见解去分析，并且距离最终的结论八九不离十。
我知道，自己的知识框架基本上搭建成功，成网状结构存生存在我的脑袋中。
新的知识到来，我会先衡量其是否正确，洞察其内在逻辑。有用的吸收到体系，无用的随手扔掉，不在理会。
当我意识到此事后，着实高兴了良久：此后，习得屠龙记，世间不迷茫。

我的知识框架搭建比较粗犷，分别使用了如下的方式：

  • 侧面看问题：必杀技，在别人讲故事的时候，从一些细小的细节发现真理。
  • 分析现实问题：将身边发生的事情，用自己的思维复盘。若是自己在当时，应该如何处理会更有利益。
  • 阅读大量“杂书”：常年挂在天涯等各种论坛，看各种有用无用的杂书，从中自己窥探真理。
  • 抽离中心思想：将事物从具体抽离出为抽象中心，纳入体系。
  • 突破条条框框的束缚：从不care各种人为指定的规则，按照自己内心的逻辑出发，处理事物。
  • 思考别人这样思维、做事的内在逻辑：假定所有人都是理性人，看他们为什么这样处理问题，我是否也能这样。

以上各条潜移默化到生活中，长期泡在手机中，方的始成。
